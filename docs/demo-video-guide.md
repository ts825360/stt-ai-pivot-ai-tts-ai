# CoolPath AI 사연영상 촬영 가이드

## 1. 촬영 목표

이번 시연영상의 목표는 CoolPath AI가 단순 지도 앱이 아니라, 여행 전 일정 설계부터 여행 중 길안내까지 이어지는 모바일 여행 비서라는 점을 보여주는 것입니다.

영상에서는 너무 많은 기능을 한 번에 설명하기보다 아래 흐름이 자연스럽게 보이는 것이 중요합니다.

1. AI 플랜 생성
2. 추천 경로 확인
3. 카메라 길안내 실행
4. 버스 정류장, 지하철 입구, 계단 같은 헷갈리는 지점 안내

핵심 메시지는 다음과 같습니다.

> CoolPath AI는 여행자가 길에서 실제로 헷갈리는 순간까지 카메라 HUD와 음성 안내로 도와주는 여행 비서입니다.

## 2. 공통 준비

휴대폰에서 아래 HTTPS 주소로 접속합니다.

```text
https://daffodil-kilogram-ageless.ngrok-free.dev/
```

카메라와 GPS 권한을 모두 허용합니다.

촬영 전 휴대폰 화면 밝기를 높이고, 가능하면 방해금지 모드를 켜는 것을 권장합니다.

카메라 시연은 실내 GPS 오차나 장소 환경에 영향을 크게 받기 때문에, 시연영상에서는 데모 URL을 사용합니다. 실제 후면카메라는 그대로 켜지고, HUD 안내 상황만 촬영자가 원하는 순간에 전환됩니다.

## 3. 데모 조작법

카메라 화면 중앙에 있는 HUD 아이콘을 약 1초 길게 누르면 데모 기믹이 실행됩니다.

길게 누르면 짧은 진동이 오며, 그 순간부터 데모 타이머 또는 상황 전환이 시작됩니다.

짧게 누르면 실행되지 않을 수 있으니 촬영자는 중앙 HUD 아이콘을 충분히 길게 눌러야 합니다.

## 4. 전체 권장 구성

### 4.1 Plan 탭

보여줄 장면:

1. Plan 탭 진입
2. 여행 기간, 여행 페이스, 장소 선택 화면을 짧게 보여주기
3. AI 플랜 생성 버튼 클릭
4. 생성된 추천 경로 화면 확인

전달 의도:

사용자가 여행지를 하나하나 고민하지 않아도 AI가 여행 동선을 구성해 준다는 점을 보여줍니다.

### 4.2 지도/경로 확인

보여줄 장면:

1. 추천 경로 카드
2. 추천점수
3. 예상 시간
4. 도보 예상 시간
5. 보행 거리

전달 의도:

CoolPath AI가 단순 최단거리만 보는 것이 아니라 이동 부담, 보행 부담, 쾌적성을 함께 고려한다는 점을 보여줍니다.

### 4.3 카메라 길안내

보여줄 장면:

1. 카메라 길안내 실행
2. 후면카메라 화면 위에 HUD 표시
3. 하단 음성 켜짐/꺼짐 버튼 확인
4. 중앙 HUD가 길안내를 제공하는 장면

전달 의도:

여행자가 지도만 보며 걷는 것이 아니라, 실제 주변 화면 위에서 필요한 안내를 바로 받을 수 있음을 보여줍니다.

## 5. 버스 정류장 시나리오

접속 URL:

```text
https://daffodil-kilogram-ageless.ngrok-free.dev/?camera=bus#camera
```

촬영 상황:

출발 위치는 호텔로 설정합니다. 호텔에서 나오자마자 우회전해야 하는 상황을 찍습니다.

촬영 순서:

1. 호텔 출구 또는 출발 위치에서 카메라 길안내를 켭니다.
2. 처음에는 기존 GPS/HUD 방향 안내가 그대로 표시됩니다.
3. 호텔에서 나와 우회전하는 장면을 촬영합니다.
4. 우회전 후 정류장으로 걸어가는 장면을 찍습니다.
5. 정류장 도착 안내를 띄우고 싶은 순간에 중앙 HUD 아이콘을 약 1초 길게 누릅니다.
6. 길게 누른 뒤 약 10초 후 `BUS 323 정류장 도착` 안내가 표시됩니다.
7. 정류장 표지판 또는 버스 정류장 주변을 카메라로 비춥니다.

보여줄 메시지:

정류장이 맞는지, 반대편 정류장으로 들어가는 실수를 줄여준다는 점을 보여줍니다.

## 6. 지하철 시나리오

접속 URL:

```text
https://daffodil-kilogram-ageless.ngrok-free.dev/?camera=metro#camera
```

촬영 상황:

지하철 입구로 접근하는 상황을 촬영합니다.

촬영 순서:

1. 지하철 입구로 걸어가는 장면을 촬영합니다.
2. 입구가 화면에 들어오기 전 또는 들어오는 순간 중앙 HUD 아이콘을 약 1초 길게 누릅니다.
3. 길게 누른 뒤 약 10초 후 `Metro 입구로 들어가 탑승하세요` 안내가 표시됩니다.
4. Metro 표지, 역 입구, 계단, 개찰구 방향이 보이도록 촬영합니다.

보여줄 메시지:

여행자가 헷갈리기 쉬운 지하철 입구, 노선 방향, 탑승 지점을 카메라 HUD로 확인할 수 있음을 보여줍니다.

## 7. 계단 시나리오

접속 URL:

```text
https://daffodil-kilogram-ageless.ngrok-free.dev/?camera=stairs#camera
```

촬영 상황:

내려가는 계단과 올라가는 계단을 모두 보여줍니다. 계단 이동 시간은 현장에서 직접 맞추는 것이 좋습니다.

촬영 순서:

1. 내려가는 계단 앞에서 카메라 길안내를 켭니다.
2. 내려가는 계단이 화면에 보일 때 중앙 HUD 아이콘을 약 1초 길게 누릅니다.
3. `이 계단으로 내려가세요` 안내가 표시됩니다.
4. 실제로 계단을 내려가는 장면을 촬영합니다.
5. 다시 올라가야 하는 계단 앞에 도착하면 중앙 HUD 아이콘을 다시 약 1초 길게 누릅니다.
6. `이 계단으로 올라가세요` 안내가 표시됩니다.
7. 올라가는 계단 또는 출구 방향을 촬영합니다.

보여줄 메시지:

실내, 환승, 계단처럼 GPS만으로 헷갈리는 구간도 카메라 HUD가 방향을 보완해 준다는 점을 보여줍니다.

## 8. 권장 편집 순서

1. Plan 탭에서 AI 플랜 생성
2. 지도에서 추천 경로 확인
3. 카메라 길안내 실행
4. 호텔 출발 후 우회전
5. 버스 정류장 확인
6. 지하철 입구 확인
7. 계단 내려가기와 올라가기
8. 마지막 문장 자막

마지막 자막 예시:

```text
CoolPath AI는 여행 전 일정 설계부터 여행 중 길안내까지 함께하는 모바일 여행 비서입니다.
```

## 9. 촬영 주의사항

카메라와 GPS는 HTTPS 주소에서만 정상적으로 허용됩니다.

카메라가 켜지지 않으면 새로고침 후 권한을 다시 허용합니다.

카메라 화면 중앙 HUD 아이콘은 짧게 누르는 것이 아니라 약 1초 길게 눌러야 합니다.

버스와 지하철 시나리오는 중앙 HUD를 길게 누른 뒤 약 10초 후 안내가 전환됩니다.

계단 시나리오는 중앙 HUD를 길게 누를 때마다 `내려가기`에서 `올라가기`로 전환됩니다.

실내 GPS는 실제 위치가 튈 수 있으므로, 시연영상에서는 데모 URL을 사용하는 것이 가장 안정적입니다.
